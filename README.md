# who dis?

i'm brian, aka ironprogrammer, a full stack dev, wordpress core contributor, plugin hacker, test and quality fiend, and automator of things. i've worked for [automattic](https://automattic.com/) as a full-time sponsored contributor, and for over a decade helped run a [marketing agency](https://oakcreektrail.com) implementing bespoke wordpress and digital marketing services and automations on all our clients. there's also other stuff over in [my linkedin](https://www.linkedin.com/in/brianalexander/).

to dig more into my wordpress open source work, head to [my wporg profile](https://profiles.wordpress.org/ironprogrammer/) or look me up in [core trac](https://core.trac.wordpress.org/search?q=props+ironprogrammer&noquickjump=1&changeset=on&ticket=on). i've [contributed to core since 5.9](https://peterwilson.cc/wordpress-contribution-history/?wp-credit-history-username=ironprogrammer), served as test team rep, joined release squads, and posted a bit in the [core](https://make.wordpress.org/core/author/ironprogrammer/) and [test](https://make.wordpress.org/test/author/ironprogrammer/) blogs.

i'm still in search of my next adventure for a great work/culture fit. check out [my website](https://brianalexander.com/) to learn more about my irl side.
